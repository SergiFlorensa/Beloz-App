# Laboratorio Python de Innovación

Este directorio recogerá los prototipos y experimentos de IA en Python (modelos, notebooks, scripts). Cada experimento deberá documentarse y, cuando esté listo, se trasladará una versión optimizada al entorno Kotlin/Android para ejecutar on-device.

Estructura propuesta:
- `prototipos/`: scripts rápidos, notebooks y datasets sintéticos.
- `modelos_exportados/`: versiones convertidas (ONNX/TFLite) listas para integrarse en la app.
